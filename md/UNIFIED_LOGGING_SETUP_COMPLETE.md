### Search with Regex
```bash
./aggregate-all-logs.sh search "ERROR|WARN"          # OR logic
./aggregate-all-logs.sh search "ERROR.*connection"   # Sequence
./aggregate-all-logs.sh search "[0-9]{4}-02-[0-9]{2}" # Dates
```

### Count Occurrences
```bash
grep "ERROR" logs/system-all.log | wc -l
grep "DATABASE" logs/system-all.log | wc -l
```

### Export for Analysis
```bash
# Extract to CSV-like format
grep "ERROR" logs/system-all.log | cut -d']' -f1-3

# Export timestamps
grep "CACHE" logs/system-all.log | cut -d']' -f1
```

### Archive & Backup
```bash
cp logs/system-all.log logs/backup-$(date +%Y-%m-%d).log
gzip logs/backup-*.log
```
